# 📚 Smart Student Management System

A console-based Python application designed to efficiently manage student records in a small academic environment. This project demonstrates core programming concepts including OOP, file handling, data structures, and modular functions—making it a strong addition to my programming portfolio.

---

## 📌 Project Details

🎯 **Project Name:** Smart Student Management System
🚀 **Tech Stack:** Python (OOP, File Handling, Data Structures)
🔗 **Role:** Python Developer (Beginner Level Project)
👨‍💻 **Goal:** To build a functional system for managing student data, marks, grades, and academic performance in an organized and automated way.

---

## 🚀 Features

✔️ Add new student records
✔️ View all student details
✔️ Search student by ID or Name
✔️ Update student marks
✔️ Calculate total marks, average, and grade
✔️ Display top-performing student (Topper)
✔️ Delete student records
✔️ Password-protected access
✔️ File handling for permanent data storage
✔️ Menu-driven interactive console system
